# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player gameplay
- Winner detection
- Draw detection
- Scoreboard
- Dark Mode / Light Mode
- Reset Game
- Reset Scores

## Technologies Used

- HTML
- CSS
- JavaScript
